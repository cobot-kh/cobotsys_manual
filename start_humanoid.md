# start humanoid


상위기 = 비전 + 의사결정 모듈 포함  
기능 설명: 로봇의 호스트 컴퓨터는 머리에 장착하는 NUC(네트워크 통합 제어 장치)로, 음성 상호 작용 및 시각적 특징 감지와 같은 이미지 및 오디오 처리 및 구문 분석을 담당합니다. 호스트 컴퓨터는 네트워크 케이블을 통해 슬레이브 컴퓨터에 연결되며, 슬레이브 컴퓨터에서 할당한 IP 주소를 자동으로 할당받습니다. 로봇의 ROS 마스터-슬레이브 시스템에서 호스트 컴퓨터는 슬레이브 역할을 합니다.  

하위기 = 제어 + 실행 스크립트 포함  
기능 설명: 하위 레벨 로봇은 흉강 내에 위치한 NUC로, 역기구학 및 보행 알고리즘과 같은 로봇의 전반적인 동작 제어를 담당합니다. 하위 레벨 로봇은 DHCP 서비스가 설정되어 있으며, 네트워크 케이블을 통해 상위 레벨 로봇에 연결되고, 상위 레벨 로봇에 IP 주소를 할당합니다. 로봇의 ROS 마스터-슬레이브 시스템에서 하위 레벨 로봇은 마스터 역할을 합니다.



** 참고: 하위 레벨 시스템 환경에서의 모든 컴파일 작업은 관리자 계정으로 수행 **
